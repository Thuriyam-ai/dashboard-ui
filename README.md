# frontend-boilerplate

Template for Frontend

## Getting Started

Initialise node environment:

```bash
nvm install && nvm use
npm install
```

Run development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
